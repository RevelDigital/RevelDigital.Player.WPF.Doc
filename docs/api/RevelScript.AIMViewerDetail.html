Ethnicity

Defines the ethnicity of an individual. (not currently functional). For age, a value of 0 represents an unknown age, 1 represents a child (under 16), 3 represents a young adult (16-34), 4 represents an older adult (35-64), and 5 represents a senior (65+). A value of 2 represents a teenager (13-19) but this age class is not currently output by the software.
