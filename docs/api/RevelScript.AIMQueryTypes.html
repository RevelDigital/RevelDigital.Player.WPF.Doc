Intel Audience Impression Metric (AIM) API Query Types

public enum AIMQueryTypes

Fields

GetAudienceDetails = 1

Requests immediate status information from AIM View regarding details of all viewers currently in front of the display.

GetAudienceStatus = 0

Requests immediate status information from AIM View regarding number of viewers currently in front of the display.

GetMajorityGender = 2

Requests the majority gender of audience members from AIM View over the past 15 seconds.

GetViewerEvents = 3

Requests continuous information from AIM View regarding number of viewers currently in front of the display. Events are sent each time a viewer starts or stops looking at the display, or when another significant visual change occurs (such as a viewer changing directions).
